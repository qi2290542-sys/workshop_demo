# 三服务部署启动步骤

在服务器上依次启动以下三个服务即可。三个服务均已绑定 `0.0.0.0`，可直接对公网提供。

## 前置

- Windows + Python 3.9+（安装时勾选 Add to PATH）
- 在防火墙 / 云安全组放行端口：`8000`、`9527`、`8010`

## 服务一：降雨量计算服务（端口 8000）

```powershell
cd 降雨量计算服务
pip install -r requirements_server.txt
python server.py
```

也可双击 `start_server.bat` 启动。

## 服务二：闸门逆向推演计算服务（端口 9527）

```powershell
cd 闸门逆向推演计算服务
pip install -r requirements_server.txt
python sluice_server.py
```

也可双击 `start_sluice.bat` 启动。

## 服务三：闸门信息页服务（端口 8010）

```powershell
cd 闸门信息页服务
python page_server.py
```

无需安装依赖（纯 Python 标准库）。也可双击 `start_page.bat` 启动。

## 验证

浏览器分别访问以下地址，能正常打开即启动成功（`<IP>` 换成服务器公网地址）：

- http://<IP>:8000/docs
- http://<IP>:9527/docs
- http://<IP>:8010/gate_info.html
